# Pokedex App

## Description

Pokedex App is a responsive frontend application created with React.js. The application uses PokeAPI to display the first 151 Pokémon and provides search, type filtering and pagination functionality.

## Features

* Responsive Pokémon cards;
* Pokémon avatar, id, name, types and main stats;
* Search by Pokémon name;
* Filtering by multiple Pokémon types;
* Pagination;
* Items per page selector: 10 / 20 / 50;
* Loading, error and empty states;
* Responsive layout for desktop, tablet and mobile devices;
* ESLint configuration.

The first stable version of the Pokedex loads the first 151 Pokémon. The Pokémon limit can be changed in the project constants if needed.
