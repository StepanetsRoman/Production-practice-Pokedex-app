# Pokedex App

## Description

This is a responsive frontend Pokedex application created with React.js. The application uses PokeAPI to display the first 151 Pokémon with search, type filtering and pagination.

## Features

- Pokémon list displayed as responsive cards;
- Pokémon avatar, id, name, types and main stats;
- Search by Pokémon name;
- Filtering by multiple Pokémon types;
- Pagination;
- Items per page selector: 10 / 20 / 50;
- Loading, error and empty states;
- Responsive layout;
- ESLint configuration.

The first stable version of the Pokedex loads the first 151 Pokémon. The Pokémon limit can be changed in `src/constants/pokemonTypes.js` via the `POKEMON_LIMIT` constant.
